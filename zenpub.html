For Publishing

ZenPub gives users the ability to anonymously publish data, documents, and media in a way that is resistant to censorship.

The purpose is ZenPub is to allow individuals to freely publish data or information without fear of reprisal by allowing the individuals to publish from a shielded address (anonymous) via the InterPlanetary File System.
